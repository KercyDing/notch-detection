# Notch Detection

Our notch-detection GUI written by zig with [dvui](https://github.com/david-vanderson/dvui) framework.

> Easy to write, fun to use.

## Build from source

- Require:
  - **Zig** ([0.16.0](https://ziglang.org/download/#release-0.16.0))
- Maybe(Written by me):
  - **Only** ([repo](https://github.com/KercyDing/only))

Run:
```bash
zig build run
# or:
# only r
```

## Why Zig?

C/C++ is boring, Rust is difficult, Python is too slow.

Zig is modern C, and it's explicit to write.

That's it.
